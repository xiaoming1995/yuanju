# 设计文档：神煞体系全面扩充

## 架构决策

### 神煞极性返回方式

**方案A**：返回 `[4][]string`（当前），极性信息通过独立查询表在前端查询  
**方案B**：返回 `[4][]ShenShaItem{Name, Polarity}`，结构更丰富但需改 JSON 格式

**选择方案A**：不改变 API 格式，极性通过独立导出的 `ShenShaPolarity` map 供前端复用，零破坏性变更。

## 新增神煞详细规则

### 1. 天德贵人（月支 → 当柱天干）

月支 → 贵人天干：
- 正月寅 → 丁 | 二月卯 → 申 | 三月辰 → 壬 | 四月巳 → 辛  
- 五月午 → 亥 | 六月未 → 甲 | 七月申 → 癸 | 八月酉 → 寅  
- 九月戌 → 丙 | 十月亥 → 乙 | 十一月子 → 庚 | 十二月丑 → 己

注：月支对应的"贵人"是**天干**（6个）或**地支**（6个）混合，天干型与地支型分别放入天干柱和地支柱。

### 2. 天德合（天德贵人的六合干）

天德贵人为天干时，其六合干即为天德合：
丁→壬，申(无合干)，壬→丁，辛→丙，亥(无合干)，甲→己，癸→戊，寅(无合干)，丙→辛，乙→庚，庚→乙，己→甲

### 3. 月德贵人（月支 → 当柱天干）

- 寅午戌月 → 丙 | 申子辰月 → 壬 | 亥卯未月 → 甲 | 巳酉丑月 → 庚

### 4. 月德合（月德贵人的六合干）

丙合辛，壬合丁，甲合己，庚合乙

### 5. 金舆贵人（日干 → 四柱地支）

甲→辰，乙→巳，丙→未，丁→申，戊→未，己→申，庚→戌，辛→亥，壬→丑，癸→寅

### 6. 天喜星（年支 → 四柱地支）

- 申子辰年 → 戌 | 寅午戌年 → 辰 | 亥卯未年 → 丑 | 巳酉丑年 → 未

### 7. 红艳（日干 → 四柱地支）

甲→午，乙→午，丙→寅，丁→未，戊→辰，己→辰，庚→戌，辛→酉，壬→子，癸→申

### 8. 天厨贵人（日干 → 四柱地支）

甲→巳，乙→午，丙→寅，丁→酉，戊→申，己→酉，庚→亥，辛→子，壬→寅，癸→卯  
（与文昌接近但不完全相同）

### 9. 三奇贵人（天干组合）

- 天上三奇：同时含甲+戊+庚（各天干位出现）
- 地下三奇：同时含乙+丙+丁
- 人中三奇：同时含壬+癸+辛  
（在含有其组合的柱上标注）

### 10. 飞刃（日干 → 四柱地支，羊刃对冲）

甲→酉，乙→戌，丙己→子，丁戊→丑，庚→卯，辛→辰，壬→午，癸→未

### 11. 福星贵人（年干 → 四柱地支）

甲→寅，乙→丑子，丙→子亥，丁→酉，戊→申，己→未午，庚→巳，辛→辰，壬→卯寅，癸→丑

### 12. 天医（月支 → 四柱地支，月支前一辰）

月支前一位地支为天医：子→亥，丑→子，寅→丑，卯→寅，辰→卯，巳→辰，午→巳，未→午，申→未，酉→申，戌→酉，亥→戌

## Bug 修复：天罗地网

**当前**：只检查年支作为基准  
**修复**：扫描全盘四柱地支，若同时存在「戌」和「亥」则所有含这两个字的柱位标记「天罗」，若同时存在「辰」和「巳」则标记「地网」。

```go
    "天德贵人": "ji", "月德贵人": "ji", "天德合": "ji", "月德合": "ji",
    "金舆贵人": "ji", "天喜星": "ji", "天厨贵人": "ji", "国印贵人": "ji",
    "三奇贵人": "ji", "日德": "ji", "将星": "ji", "福星贵人": "ji", "天医": "ji",

    // 凶
    "羊刃": "xiong", "劫煞": "xiong", "亡神": "xiong",
    "孤辰": "xiong", "寡宿": "xiong", "阴差阳错": "xiong",
    "魁罡": "xiong", "十恶大败": "xiong", "天罗": "xiong", "地网": "xiong",

    // 中性（需结合格局）

    "天医": "ji",
}
```

## 前端渲染变化

`ResultPage.tsx` 神煞标签根据极性加 CSS class：
- `ji` → 金色/暖色（`shensha-tag--ji`）
- `xiong` → 红色/冷色（`shensha-tag--xiong`）
- `zhong` → 灰色（默认）
