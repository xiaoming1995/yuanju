# 神煞注解系统（shensha-annotations）

## 背景与目标

目前缘聚平台的基本排盘已能正确计算 40+ 个神煞（如天乙贵人、天罗地网、词馆等），但用户在看到这些标签时，无法理解其含义。本变更为每个神煞增加详细解说文案，通过点击标签触发浮层卡片展示，提升命理知识的可读性与平台专业度。

## 范围决策

### Phase 1（本次实现）
- 后端新增 `shensha_annotations` 数据表
- 管理员可通过 API 修改神煞描述（后台 UI 为 Phase 2）
- 公开 REST API：`GET /api/shensha/annotations`，返回全部神煞注解
- 前端结果页：点击神煞标签 → 弹出浮层卡片（含名称、极性、详细描述）
- 后端预置 40+ 个神煞的默认文案（命理书级别详细说明）

### Phase 2（后续实现，已记录）
- 管理后台增加「神煞注解管理」页面，支持在线编辑更新文案
- 神煞分类标签系统（贵人系/桃花系/格局限制等）
- 桌面端 hover 3秒预览 + 点击固定浮层双模式

## 用户场景

```
用户查看 1996年命盘结果页
└─ 看到月柱有 [墓门] 标签
   └─ 心里疑问：这是什么？对我有什么影响？
      └─ 点击 [墓门]
         └─ 弹出卡片：
            ┌─────────────────────────────┐
            │ 🔴 墓门                      │
            │ ─────────────────────────── │
            │ 墓门者，天干五行克地支五行    │
            │ 之局...（详细说明 100-200字）│
            │                    [关闭 ×] │
            └─────────────────────────────┘
```

## 非目标

- 不做个性化解读（所有用户看同一神煞看到相同文案）
- 不在 Phase 1 做 Admin 编辑 UI
- 不改变现有神煞计算逻辑

## 成功标准

- 所有已实现神煞均有对应注解文案
- 点击任意神煞标签均能触发浮层卡片
- 浮层卡片在移动端和桌面端均可用
- Admin 可通过 API 更新文案
