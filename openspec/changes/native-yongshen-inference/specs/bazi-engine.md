# Capability: Bazi Engine (Modified)

**Overview:**
此能力规范控制整个八字排盘的系统命核，涵盖从时间转换为干支到底层五行成分解构的功能。

## Native Yongshen Inference (新增算法)
- 当处理日柱并导出属性分配后，判定五行的生、克状态。
- 如果属于“身强”阵营（即生扶自身的两项属性，其合集比例不小于 40%）：判定命理偏激需打压。
  - **喜用神（Yongshen）**：克制、泄耗命主的三个五行元素。
  - **忌神（Jishen）**：生扶命主的两个五行元素。
- 反之若为“身弱”阵营。
  - **喜用神（Yongshen）**：生扶自身的两项属性。
  - **忌神（Jishen）**：克泄自身的三个五行元素。
- 输出的值直接内联在返回给接口的最底层 `BaziResult` 结构中。
