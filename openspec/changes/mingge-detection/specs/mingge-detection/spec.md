## ADDED Requirements

### Requirement: 七优先级透干取格
系统 SHALL 按以下七个优先级依序判定命格，取第一个命中的结果：
1. 月支主气藏干透出到月干（月柱自透）
2. 月支任意藏干透出到年干/日干/时干（月支透它干）
3. 其它三柱地支主气藏干透出到月干（它支透月干）
4. 其它三柱地支任意藏干透出到任意天干（它支透它干）
5. 同优先级内多个格神时，取透出次数最多者；次数相同则取五行力量最强者
6. 无透格神时，检测四柱是否构成三合局或三会局，取合局五行对应十神为格神
7. 兜底：统计全局干支中各十神出现总数，满四个以上者取最高频者成格

#### Scenario: 月柱自透命局
- **WHEN** 月支藏干之主气与月干相同（如月支寅，主气甲，月干亦为甲）
- **THEN** 系统返回该天干对应日主十神关系 + "格"后缀作为 ming_ge

#### Scenario: 月支透它干命局
- **WHEN** 月支藏干中某干在年干/日干/时干中出现，但月干不属于月支藏干
- **THEN** 系统返回该藏干对应日主十神关系 + "格"后缀作为 ming_ge

#### Scenario: 无透格神时有三合局
- **WHEN** 前四优先级均未命中，且四柱地支构成三合局（如寅午戌合火局）
- **THEN** 系统以合局五行（火）对应日主十神关系 + "格"后缀作为 ming_ge

#### Scenario: 七优先级全部未命中
- **WHEN** 七个优先级均无法确定格局
- **THEN** 系统返回 ming_ge 为 "杂气格"，ming_ge_desc 为兜底说明文字

### Requirement: 建禄格与月刃格特殊映射
系统 SHALL 在格神对应十神为比肩时返回"建禄格"，十神为劫财时返回"月刃格"，而非直接使用十神名称。

#### Scenario: 月令比肩格神
- **WHEN** 取格命中的格神（天干）相对日主的十神关系为"比肩"
- **THEN** 系统返回 ming_ge 为 "建禄格"

#### Scenario: 月令劫财格神
- **WHEN** 取格命中的格神（天干）相对日主的十神关系为"劫财"
- **THEN** 系统返回 ming_ge 为 "月刃格"

### Requirement: 格局说明文字字典
系统 SHALL 为每个标准格名内置固定的中文说明文字（ming_ge_desc），随 API 响应返回。

#### Scenario: 标准格名有说明文字
- **WHEN** ming_ge 为十大标准格名之一（正官格/七杀格/食神格等）
- **THEN** ming_ge_desc 返回对应格局的命理简介，不少于 50 字

#### Scenario: 非标准格名
- **WHEN** ming_ge 为"杂气格"
- **THEN** ming_ge_desc 返回通用兜底说明文字
