# Capability: Bazi UI (Modified)

**Overview:**
控制排盘数据的全面渲染，确保排版能接住新扩展的十神架构。

## 专业排盘卡片增强
- **天干附加信息：** 每柱的天干顶部必须附带其“十神”（ShiShenGan）。对于日柱而言，此处为“日主”或“元女/男”。
- **地支附加信息：** 在地支下方（或者藏干下方）标注该地支主气的十神关系（ShiShenZhi）以及对应的地势长生十二宫（DiShi）。
- **样式统一：** 这类补充信息的展示应该利用静默色块（如 `var(--text-secondary)`），不要喧宾夺主抢走天干地支主体字体的视觉中心。
